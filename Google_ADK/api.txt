Keep adk web --port 8080 running when using below APIs:

Retrieve specific session id for user
GET http://localhost:8080/apps/google_search_agent/users/cskhachane/sessions/{session_id}
List all sessions for user
GET http://localhost:8080/apps/google_search_agent/users/cskhachane/sessions
Create specific session_id for user
POST http://localhost:8080/apps/google_search_agent/users/cskhachane/sessions/{session_id}
Create new session with automatic session_id for user
GET http://localhost:8080/apps/google_search_agent/users/cskhachane/sessions
Deletes specific session id
DELETE http://localhost:8080/apps/google_search_agent/users/cskhachane/sessions/{session_id}
